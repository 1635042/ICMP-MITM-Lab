# ICMP Redirect MITM Attack Lab

This project demonstrates an ICMP redirect attack to establish a Man-in-the-Middle (MITM) position and modify network traffic.

## Tasks
- Task 1: Network Setup
- Task 2: ICMP Redirect Attack
- Task 3: MITM Eavesdropping
- Task 4: Traffic Modification

## Technologies
- Docker
- Scapy
- NetfilterQueue
- tcpdump
